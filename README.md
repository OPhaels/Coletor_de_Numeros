Programa simples com interface gráfica montada para suprir uma necessidade: Remover números de imagens e transferí-los para um campo de texto que possa ser copiado, ignorando as letras. Não é armazenado informações, apenas serve para a coleta de números de imagens. As cores da imagem são removidas e com a ajuda dos filtros conseguimos identificar números.
